# UNIQ — Текущий путь лида

Интерактивная диаграмма процесса работы с лидами для UNIQ.

## Описание

Визуализация текущего пути лида с возможностью просмотра деталей каждого этапа через модальные окна.

## Файлы

- `uniq_flowchart_preview_html.html` — основной HTML файл с диаграммой

## Использование

Откройте `uniq_flowchart_preview_html.html` в браузере.
